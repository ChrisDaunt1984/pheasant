5. Macro

Default macro definition syntax is

[=tag]: **replaced text**

In this case, all occurence of '[=tag]' are replaced by 'replaced text'

You can use inline code like this:

[=2x4] {{2*4}}

Then, '[=2x4]' are replaced by '8'
