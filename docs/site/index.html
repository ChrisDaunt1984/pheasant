Pheasant

Welcome to Pheasant! In this section, overview and some interesting features of Pheasant are presented.

Overview

Pheasant is a Markdown converter which is designed to work with MkDocs as a plugin.

Highlights include:

  • Auto generation of outputs for a fenced code block or inline code in Markdown source using Jupyter client. The code language is not restricted to Python.
  • Auto numbering of headers, figures, and tables, and etc. Numbered objects can be linked from other Markdown sources.

Installation

You can install Pheasant from PyPI.

$ pip install pheasant

If you use Pheasant as a plugin of MkDocs, you also need to install it.

$ pip install mkdocs

Settings

In your mkdocs.yml, add lines below:

theme:
  name: rtd-pheasant

plugins:
  - pheasant

Getting Started

Auto generation of the executed outputs with Jupyter client

A markdown source below:

```python
print(1)
```

is converted into:

```python
print(1)
```

```python
1
```

after execution of print function via Jupyter client and finally rendered as:

```python .pheasant-fenced-code .pheasant-input print(1)

```python .pheasant-fenced-code .pheasant-output
1

Inline code embeded in a Markdown source

"Inline code" is a powerful feature of Pheasant. Any python codes surrounded by {{ and }} are executed and the result remains there. For example, {{3*5}} becomes 15. Variables can be assigned in an inline code: {{name='Pheasant'}}. Then, "I'm {{name}}." becomes "I'm Pheasant." Note that an inline code without outputs is not shown after execution.
